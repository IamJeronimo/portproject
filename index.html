Dynamic Space

O Dynamic Space é um projeto desenvolvido pela equipe SpaceLink, com integrantes dos cursos de Redes de Computadores e Desenvolvimento de Sistemas da instituição de ensino SENAI Informática. O projeto consiste em uma aplicação automatizando a criação de ambientes na nuvem pública da Microsoft, foi desenvolvido para a empresa colaboradora Space Needle.

UniEats

A UniEats foi um projeto acadêmico da Universidade Nove de Julho, visando melhorar o atendimento em restaurantes evitando o contato entre os funcionários e os clientes tendo em vista que em 2020 vivenciamos uma pandemia. O projeto consiste em um sistema Java Web, onde o cliente pode fazer o seu pedido acessando do seu próprio SmartPhone, também é possível ter informações atualizadas sobre seu pedido.

SportWear

O projeto SportWear foi o realizado no primeiro periodo do curso de Ciências da Computação na Universidade Nove de Julho, neste projeto criamos uma empresa fictícia a "SportWear", uma rede de produtos esportivos como tênis, calças e camisetas para corrida. A ideia principal seria criar um web site para a SportWear a qual neste cenário possuia apenas lojas físicas, porém com a crescente demanda e a evolução do mercado resolveu ter o seu próprio web site.
